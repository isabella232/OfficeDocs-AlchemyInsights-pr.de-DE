# <a name="cant-download-files-in-chrome"></a>Herunterladen von Dateien in Chrome nicht möglich

**Problem** Wir kennen ein Problem, das verhindert, dass einige Kundendateien herunterladen können, wenn Sie Google Chrome Browser verwenden. 

**Problemumgehung** [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) oder einen anderen Webbrowser verwenden.
Erfahren Sie mehr über das [Herunterladen von Dateien und Outlook.com-Anlagen](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy).

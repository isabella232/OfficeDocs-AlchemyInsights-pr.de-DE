# <a name="public-folder-access-from-mobile-devices"></a>Zugriff auf Öffentliche Ordner von mobilen Geräten

Wenn Sie ähnliche Funktionen wie öffentliche Ordner haben möchten und Inhalte auf mobilen Geräten zugänglich sind, finden Sie weitere Informationen [zu Office 365 Gruppen](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) für eine bessere Alternative.

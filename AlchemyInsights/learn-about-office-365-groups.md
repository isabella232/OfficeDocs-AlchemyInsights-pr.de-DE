# <a name="learn-about-microsoft-365-groups"></a>Informationen zu Microsoft 365-Gruppen

Microsoft 365 Gruppen ist ein Dienst, der mit den bereits verwendeten Office 365 Tools zusammenarbeitet, sodass Sie beim Schreiben von Dokumenten, Erstellen von Tabellenkalkulationen, Arbeiten an Projektplänen, Planen von Besprechungen oder Senden von E-Mails mit Ihren Teamkameraden zusammenarbeiten können. Weitere Informationen finden Sie unter ["Informationen zu Microsoft 365-Gruppen".](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2)
